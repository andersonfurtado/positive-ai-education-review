# Positive AI in Education ‚Äì Systematic Review (2010‚Äì2025)

This repository contains the supplementary data, materials, and protocols used in the systematic review *‚ÄúToward Positive AI in Education: Integrating Artificial Intelligence, Positive Psychology, and Socio-Emotional Learning‚Äù*.

The study followed **PRISMA 2020 guidelines** to ensure transparency, reproducibility, and methodological rigor.

## üìÇ Repository Structure
- **data/** ‚Üí Consolidated search results, PRISMA flow counts, excluded records.
- **materials/** ‚Üí Search strings, PRISMA checklists, and extraction form.
- **results/** ‚Üí Descriptive statistics, included study summaries, and figures.
- **docs/** ‚Üí Manuscript versions and supplementary methodological notes.

---

## üìä Data Availability
The dataset includes:
- Consolidated database of search results (`tb_bases_consolidadas_2025_09_01_FINAL.xlsx`);
- Full search strings adapted for each database (Scopus, Web of Science, PsycINFO);
- PRISMA 2020 flow diagram counts and checklist;
- Data extraction form (Excel).

Due to copyright restrictions, **full-text articles are not shared**. Metadata and extraction tables are provided to enable replication.
